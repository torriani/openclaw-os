# Etapa 0 — Abertura

**Tempo:** 18 min
**Objetivo:** Aluno entende o que vem, valida pre-reqs e ta pronto pra Etapa 1.

## Conteudo da etapa

### Step 1: Por que estamos no momento certo

Mostre exatamente isto:

```
========================================
  ETAPA 0 — ABERTURA
========================================

Por que voce esta no momento certo pra construir um agente AI pessoal?

Tres convergencias acontecendo agora:

  1. MODELOS comoditizados — GPT-5, Claude 4, Gemini 2.5
     todos ja chegaram em qualidade comparavel a um humano competente.

  2. INFRAESTRUTURA barata — VPS Hostinger sai R$ 49/mes.
     Em 2020 isso custava 10x mais.

  3. INTERFACE ja existe — Telegram. Voce ja usa todo dia.
     Nao precisa app proprio, site, login.

A janela de quem construir AGORA e quem so consumir depois esta se fechando.
Voce vai sair desta etapa com a base mental pra construir o seu.
```

### Step 2: Analogia Matrix — as 8 etapas

```
Lembra do Neo no Matrix carregando "Eu sei kung fu" direto pro cerebro?

Vamos fazer parecido com seu agente. Em vez de TREINAR ele do zero
(meses de trabalho), vamos CARREGAR 8 skills nele em 5 horas:

  Skill 1: Setup       — corpo (VPS rodando)
  Skill 2: Seguranca   — sistema imunologico bruto
  Skill 3: Identidade  — alma, personalidade
  Skill 4: Memoria     — pra ele lembrar de tudo
  Skill 5: Integracoes — bracos (Calendar, Gmail, APIs)
  Skill 6: Proatividade — pulmao (respira sozinho)
  Skill 7: Imunologico — defesa avancada
  Skill 8: Use Cases   — repertorio de acoes prontas

Cada skill e uma etapa que voce passa nesta jornada.
Sem pular etapa. Sem inverter ordem. Quem pula trava.
```

### Step 3: Visualizar o resultado final

```
No final das 8 etapas, voce vai ter:

  - Um agente AI rodando 24/7 numa VPS sua (~R$ 49/mes)
  - Conectado ao seu Telegram (so VOCE pode falar com ele)
  - Com personalidade propria (nao e ChatGPT generico)
  - Com memoria persistente (lembra de tudo entre sessoes)
  - Integrado a Calendar, Gmail, Drive
  - Proativo (te avisa coisas, manda lembrete, roda relatorio)
  - Auto-recuperavel (se cair, ele volta sozinho)
  - Versionado no GitHub (backup automatico)

Voce vai conversar com ele pelo Telegram como conversa com um socio.
Ele tem nome, personalidade, opiniao propria. Pode discordar de voce
com argumento.

Custos totais: ~R$ 150/mes (VPS + ChatGPT Plus).
Tempo de setup: ~5h, parcelado em 1-3 sessoes.
```

### Step 4: Voce se encaixa no perfil?

```
Esta jornada faz sentido pra voce se voce e:

  ✓ Empreendedor / dono de negocio (faturamento R$ 50k+/mes)
  ✓ Criador de conteudo (audiencia e/ou produtos digitais)
  ✓ Profissional liberal (advogado, contador, consultor, medico, terapeuta)
  ✓ Dev / tech ja familiar com terminal

NAO e pra voce se:
  ✗ Voce nunca abriu um terminal e tem zero paciencia pra aprender
  ✗ Voce quer "agente AI gratis pra brincar" — isso aqui e ferramenta seria
  ✗ Voce nao consegue comprometer 5h pro setup

Se voce ainda nao tem certeza, melhor parar agora. Quem segue, segue ate o
final.

Voce esta dentro?
```

Aguarde "sim" ou similar. Se "nao", oriente: "Sem problema. O setup vai estar
aqui quando estiver pronto." e PARA.

### Step 5: Validar 3 pre-requisitos

```
Antes da Etapa 1, confirma que voce tem os 3 pre-requisitos:

  1. ChatGPT Plus contratado
     - Onde: chatgpt.com → Upgrade
     - Custo: $20/mes
     - Por que: vamos usar OpenAI Codex 5.5 como modelo principal,
       voce precisa da conta paga pra ter acesso

  2. Telegram instalado e funcionando
     - Onde: App Store / Play Store
     - Custo: gratis
     - Por que: e o canal de comunicacao com seu agente

  3. VPS Hostinger contratada
     - Onde: hostinger.com.br/vps-hosting
     - Plano: KVM 2 (recomendado)
     - Custo: R$ 49/mes (12 meses)
     - Sistema: Ubuntu 24.04
     - Por que: onde seu agente vai rodar 24/7

Voce tem os 3?
```

Aguarde resposta. Se faltar:

- **Sem ChatGPT Plus:** "Vai em chatgpt.com agora, contrata, e volta. Sem ele a Etapa 1 nao roda."
- **Sem Telegram:** "Baixa no celular, cria conta, volta. 2 minutos."
- **Sem VPS:** "Vai em hostinger.com.br → VPS Hosting → KVM 2 → Ubuntu 24.04. Anota IP e senha root quando terminar de provisionar. Pode demorar 5-10min pra ficar pronto. Volta aqui depois."

So avanca quando todos OK.

### Step 6: Mentalizar o formato

```
ULTIMO PONTO antes da Etapa 1:

Esta jornada NAO e teorica. E pratica de verdade.

  - Voce vai abrir terminal
  - Voce vai colar comandos
  - Voce vai criar arquivos no seu computador
  - Voce vai mandar prompts pro Telegram
  - Voce vai ver coisas darem certo (e algumas darem errado)

Eu vou te guiar passo a passo. Nao vou pular nada.

Tem alguma duvida AGORA, antes de comecar?

(Se nao tiver, responde "vamos" que eu te libero pra Etapa 1)
```

Aguarde "vamos" ou similar.

## Veto conditions

- Algum dos 3 pre-reqs faltando → PARA. Nao avanca pra Etapa 1.
- Aluno hesita ("acho que nao preciso disso") → respeita, oriente "talvez nao
  seja a hora certa, pode voltar quando estiver pronto" e fecha.
